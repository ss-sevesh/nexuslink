## Summary

The CRISPR-Cas9 system provides adaptive immunity in bacteria against bacteriophage by using guide RNA to direct the Cas9 endonuclease to complementary DNA sequences, inducing double-strand breaks and triggering homology-directed repair.

## Key Findings

- Demonstrates CRISPR-Cas9 as a core process.
- Provides evidence for cross-domain analogies with repair and fault-tolerance systems.
